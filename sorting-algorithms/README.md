# Bubble Sort, Merge Sort and Radix Sort
 - Write a bubble sort, a merge sort, and a radix sort function that accept an array of integers and return an array that is sorted from least to greatest.
 ## All Functions
 - When you input:
    - [100, 20, 3, 8]
    - Returns:
        - [3, 8, 20, 100]
